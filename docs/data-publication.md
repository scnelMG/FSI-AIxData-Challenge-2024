# 데이터 공개 정책

## 현재 공개 저장소 기준

대회 원본 데이터와 제출 산출물은 GitHub 공개 브랜치에서 제외합니다.

제외 대상:

- `data/train.csv`
- `data/test.csv`
- `data/sample_submission.csv`
- `submissions/final_submission.zip`

## 제외 이유

데이터가 합성 금융거래 데이터로 설명되어 있더라도, 대회 데이터와 제출 산출물의 재배포 조건은 대회 약관을 우선해야 합니다. 또한 50MB를 넘는 CSV 파일은 GitHub 공개 저장소의 검토성과 유지보수성도 낮춥니다.

## 재현 안내

노트북을 재실행하려면 대회 페이지에서 데이터를 직접 내려받아 로컬 `data/` 폴더에 배치해야 합니다. 공개 저장소는 문제 정의, 검증 설계, 모델링 흐름, 회고를 평가 대상으로 제공합니다.
