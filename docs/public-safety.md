# 공개 안전성 검토

## Drive 매칭 결과

Google Drive에는 다음 성격의 파일이 확인되었습니다.

- 대회 원본 데이터와 제출 ZIP
- 실험별 submission 폴더
- CTGAN, LightGBM, XGBoost 실험 노트북
- `.git` 백업 폴더

## 공개 기준

이 저장소는 금융권 포트폴리오로 활용할 수 있도록 실험 설계와 회고를 공개합니다. 다만 원본 데이터, 제출 압축 파일, Drive 내 `.git` 백업, 반복 실험 산출물은 추가 업로드 대상에서 제외했습니다.

## 리뷰어 안내

- 대회 데이터는 합성 데이터로 설명되어 있으나, 재배포 조건은 대회 약관을 우선합니다.
- GitHub 제출 전에는 `data/`와 `submissions/` 공개 가능 여부를 별도로 확인하는 것이 안전합니다.
- 평가 대상은 README, `docs/`, 주요 노트북의 실험 설계입니다.
